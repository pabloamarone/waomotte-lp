# 和を以て LP

GitHub Pagesで公開できる静的ランディングページです。

## 構成

- `index.html`: ページ本体
- `style.css`: デザイン、レスポンシブ対応
- `script.js`: スムーススクロール補助、モバイルメニュー
- `assets/`: OGPなどの静的素材置き場

## 公開方法

1. このフォルダをGitHubリポジトリにアップロードします。
2. GitHubのリポジトリ設定からPagesを有効にします。
3. 公開元を`main`ブランチのルートに設定します。
